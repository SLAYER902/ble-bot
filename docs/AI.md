# BLE AI

BLE AI is provider-agnostic. The provided adapter speaks the OpenAI-compatible chat-completions format and is disabled by default. It uses bounded inputs, output caps, timeouts, an untrusted-content delimiter, and no direct database, shell, or Discord administration tool access.

Credits are reserved atomically before a provider call, committed once it succeeds, and refunded if it fails. The default monthly free balance is ten credits. AI output can assist explanation and drafting but must not independently ban, kick, or permanently punish a member.
