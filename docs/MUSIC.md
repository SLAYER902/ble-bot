# BLE Music

The deployment stack includes a Lavalink v4 service and the Node project includes `@discordjs/voice`. BLE Shield is isolated from music health and failures.

No player command is currently registered. A music feature must use legally permitted source adapters, queue persistence, voice recovery, DJ policy, bounded queue sizes, and an explicit Lavalink health check before it is exposed. BLE Bot does not scrape restricted sources, bypass DRM, or download unauthorized media.
